# personal-agent

Personal automation workspace split by domain.

| Path | Purpose |
|------|---------|

| `brands/Marketcircuit/` | Marketcircuit brand assets. |
| `brands/Workcircuit/` | Workcircuit brand assets. |

Run the existing Upwork workflows from `upwork/`.
